Download game files

  1. Log in to your Google Drive.

  2. Go to Shared with me.

  3. Right-click RAR file and click Download.

  4. Select DOWNLOAD ANYWAY if an alert says that Google Drive can’t scan file for viruses. A file can be over 100 MB which is the limit for scanning.


Extract game files with WinRAR

  1. Right-click the downloaded RAR file and select Extract files...

  2. Select OK.

  3. Enter this password in red and select OK. 1.pcgame

If you don't have WinRAR, get it here and install with these steps.


Install game

  1. Go inside the folder that you have extracted.

  2. Double-click an EXE file and finish the installation.

  3. Repeat the same for another EXE file.

Muat turun fail permainan

  1. Log masuk ke Google Drive anda.

  2. Pergi ke Shared with me.

  3. Klik kanan fail RAR dan klik Download.

  4. Pilih DOWNLOAD ANYWAY jika amaran mengatakan bahawa Google Drive tidak boleh mengimbas fail untuk virus. Fail boleh melebihi 100 MB iaitu batas pengimbasan.


Ekstrak fail permainan dengan WinRAR

  1. Klik kanan fail RAR yang dimuat-turunkan dan pilih Extract files...

  2. Pilih OK.

  3. Masukkan kata laluan berwarna merah ini dan pilih OK. 1.pcgame

Jika anda tidak ada WinRAR, dapatkan aplikasi ini di sini dan pasang dengan langkah-langkah ini.


Pasang permainan

  1. Pergi ke dalam folder yang anda sudah ekstrak.

  2. Klik dua kali pada fail EXE dan selesaikan pemasangan.

  3. Ulang perkara yang sama untuk fail EXE yang lain.
